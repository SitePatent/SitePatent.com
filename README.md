## `//` what this is

**[SitePatent](https://sitepatent.com)** is a curated, human-reviewed directory for startups, tools, and online products that actually deserve attention — the corner of the web where useful things stop being invisible.

The idea is in the name. When you file a patent, the record is permanent — it doesn't expire, it doesn't get quietly revoked six months later. **A SitePatent listing works the same way.** You submit once, a human reviews it inside 72 hours, and your placement — plus your backlink — stays on the record. For good.

> No bulk imports. No link farms. No subscription traps. Just a registry people actually scroll through.
